LEAN and the Logic Manual

LEAN is an open source programming language and a theorem prover. This resource introduces LEAN for Propositional Logic for users of the Logic Manual by Volker Halbach, and . It has been adapted for use at King's College London by Jordan Brown, Connor Molinski, Elena Wuellhorst, Carlo Nicolai.

Acknowledgements: for the content and some notational choices, the project is based on the Logic Manual by Volker Halbach, Oxford University Press, 2010; for implementation in LEAN, we relied on Logic and Proof by Jeremy Avigad, Robert Y. Lewis, and Floris van Doorn, and the LEAN 4 webeditor.
