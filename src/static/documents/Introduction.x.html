LEAN and the Logic Manual

Introduction To Using Lean

This section gives you the fundamental background to using LEAN. We introduce the LEAN syntax so that you can write the logical connectives. This will be particularly helpful in the next sections, where we go through the introduction and elimination rules of each connective. You will be able to verfiy your reasoning directly, being guided by the LEAN editor.

LEAN is an automated theorem prover- so if you input your deduction, it will show you errors and lead you to the correct path. We will familiarise ourselves with the basic reasoning in propositional logic, but LEAN is much more powerful. It is used for example at Imperial College by Kevin Buzzard to teach undergraduate mathematics. The tool helps to construct and verfiy proofs of difficult constructions. With this resource, you can familiarise yourself with the basics of the LEAN and learn logical deductions using a different approach.

You will have already seen the LEAN editor at the sidebar, this is where you can write LEAN. In this main page, we will do the basic introductions and you can directly test it in the editor. There will always be extra exerises to try on your own. But before, we need to learn the syntax of LEAN. How can you write the connective in the programming language?
