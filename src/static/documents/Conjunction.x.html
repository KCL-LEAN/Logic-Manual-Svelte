Conjunction

The well-known rules for conjunction in propositional logic are the following: $$\infer{\land Intro}{A\land B}{A&B}$$ $$\infer{\land Elim} {A} {A\land B}$$ $$\infer{\land Elim} {B} {A\land B}$$

In the programming languag LEAN, the rules function similarly. For example, in the LEAN syntax you write: 
and.right h
This extracts - from the hypothesis h - the right conjunct A. $h: A \land B$ Similarly, to get the left conjunct, you can use the following syntax in LEAN 
and.left h
We have to be more specific in LEAN, indicating exactly which conjunct we want to extract. We have already seen the elimination rule, now for the introduction of a conjunction of A and B we write: 
and.intro A B

and.intro hA hB
The latter formulation works for hypothesis hA, hB respectively. Next let us consider an example natural deduction proof involving both the introduction and the elimation rules for conjunction:

Example 1: $B \land A \vdash A \land B$ $$\infer{\land Intro} {A\land B} {\infer{\land Elim} {A} {B \land A} & \infer{\land Elim}{B} {B \land A}} $$

This proof is also done in LEAN; as you can see in the code editor on the page. This should give you an example to then continue the exercises in LEAN using the syntax for conjunction we just introduced. There are easier and harder exercises.
